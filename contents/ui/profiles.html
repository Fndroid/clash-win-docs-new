# Profiles 配置

TIP

深颜色表示当前配置文件已被载入成功

远程配置文件指通过 URL 导入的配置文件

每个配置文件都包括如下信息：

  • 名称
  • 类型（远程显示域名，本地显示 local）
  • 上次更新时间
  • 操作栏

操作栏分别可以进行如下操作：

  1. 文本编辑
  2. 策略编辑
  3. 规则编辑
  4. 复制配置文件
  5. 修改配置文件信息
  6. 刷新配置文件（只有远程配置文件可以进行此操作）
